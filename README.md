# Tournament Management System 🏆

Welcome to the Tournament Management System!  
This project is a full-stack web application built with Laravel and Vue.js that allows users to create and manage tournaments, teams, matches, and results through a structured and dynamic interface.

## Features ✨ <a name="features"></a>

- **Tournament Creation** 🏅 – Create and manage multiple tournaments  
- **Team Management** 👥 – Register and organize teams  
- **Match Scheduling** 📅 – Generate and manage matches

## Installation 📦 <a name="installation"></a>

To run the project locally:

### Backend
1. Navigate to the `backend` folder  
2. Run `composer install`  
3. Copy `.env.example` to `.env` and configure your database  
4. Run `php artisan key:generate`  
5. Run `php artisan migrate`  
6. Start the server with `php artisan serve`

### Frontend
1. Navigate to the `frontend` folder  
2. Run `npm install`  
3. Run `npm run dev`

---

## Usage ⚙️ <a name="usage"></a>

1. Start both backend and frontend servers  
2. Open the frontend in your browser  
3. Create tournaments and register teams  
4. Schedule matches and record results  
5. View dynamically updated standings
